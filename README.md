# Verified Carbon Credit (VCC)

ERC-20 tokenisation of verified carbon credits.
UCL IFTE0007 Decentralised Finance and Blockchain — Individual Coursework

## Contract Address (Sepolia Testnet)
`0x40b533f9423dE517CC3a03f3aA53Df484e199390`

## Token Design
- 1 VCC = 1 tonne CO₂-equivalent offset
- Fungible Token (ERC-20), 18 decimals
- Minting: authorised verifiers only
- Retirement: permanent burn with on-chain receipt
